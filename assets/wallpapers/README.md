# Wallpapers

Place your wallpaper collection here.

## Usage

The setup script will make wallpapers available, but you'll need to set them manually:

1. Right-click on desktop → "Configure Desktop and Wallpaper"
2. Click "Add Image..."
3. Navigate to this repository's `assets/wallpapers/` directory
4. Select your wallpaper

## Organization

You can organize wallpapers into subdirectories:

```
wallpapers/
├── landscapes/
├── abstract/
├── dark/
└── minimal/
```
